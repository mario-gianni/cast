the planning system uses eclipse constrained programming language compiler.
for this reason you need to download and install eclipse cpl (http://eclipseclp.org/)
compiler, follow the instructions given below.

how to install eclipse clp compiler
===================================

1. download the eclipse clp compiler from http://eclipseclp.org/Distribution/
   preferabally use the latest version (6.0_163) and download all the files
   below your matching architecture (e.g. i386_linux)
   note: for 64-bit cpu use the version 6.0_75/x86_64_linux

2. to install eclipse clp compiler follow the instructions in Readme.txt file

3. add eclipse.jar to your CLASSPATH
   (usually found under <eclipsedir>/lib)
